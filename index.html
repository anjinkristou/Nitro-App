Welcome to Nitro The app.

This is a standalone wrapper for Nitro. It works on Windows, Mac and Linux.

How to compile.

It's easy:

$ git clone --recursive https://github.com/anjinkristou/Nitro-App.git
$ cd Nitro-App
$ qmake Nitro-App.pro
$ make

Then run the Nitro-App application.

Just to use it.

You can always download the compiled version of Nitro-App with its installer. No need for environment settings and installation of dependencies. Just head to the download section.
